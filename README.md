AssetTrack Pro — Field App
AssetTrack Pro is a high-performance, mobile-optimised web interface designed for field operations, asset auditing, and real-time inventory management. It provides a streamlined workflow for field officers to capture, verify, and track physical assets in various environments.

🚀 Features
Mobile-First Design: Optimised for field use with a responsive 393x852px viewport focus.

Dual-Theme Interface: Includes "Light Sky-Blue" and "Dark Ocean" modes for high visibility in different lighting conditions.

Integrated Barcode Scanner: Built-in scanner overlay with a live video feed and framing guide for rapid data entry.

Asset Auditing Workflow: Specialised "Audit Scan Zone" with progress tracking and status verification.

Comprehensive Data Capture: Structured forms for asset descriptions, serial numbers, and photo attachments.

Performance Monitoring: Real-time dashboard showing "My Captures," "Audited" counts, and sync status.

Error Management: Built-in field error logging and recovery system.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Custom Variables for Theming)

Typography: Nunito (Headings), Arial/Helvetica (Data-heavy body text)
